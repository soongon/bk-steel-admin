## 2026-05-08 — [feature] 매입 부가세 수식 — 무자료(B계좌) 매입은 0

**변경 이유**:
- 도메인_룰 §1: B계좌는 무자료 거래 전용 → 부가세 신고 X.
- 영남펌프카 매입 (B계좌)에 부가세 73,150 / 25,000이 자동 잡혔음. 무자료 매입은 부가세 신고에 들어가지 않으므로 0이어야 함.

**변경**:
- `2.매입` N열 부가세 수식:
  - 변경 전: `=IF(M="","",ROUND(M*0.1,0))`
  - 변경 후: `=IF(M="","",IF(OR($Q="B계좌",$X="B계좌"),0,ROUND(M*0.1,0)))`
- 사업자(Q) **또는** 출금통장(X) 중 하나라도 `B계좌`면 부가세 0
- 둘 다 정상 (법인/사업자 + 법인A/사업자A)이면 기존 10%

**효과 (현재 데이터)**:
- 영남펌프카 #1: 부가세 73,150 → 0, 합계 804,650 → 731,500
- 영남펌프카 #2: 부가세 25,000 → 0, 합계 275,000 → 250,000
- 엠에스스틸 ×3: 변경 없음 (사업자A, 정상거래)

**통장 미러링 정합성 회복 (부수 효과)**:
- 통장 영남펌프카 출금 행은 731,500 / 250,000 으로 첫 미러링됨
- 그 후 부가세 자동계산으로 합계 변경됐으나 미러링은 INSERT-only
- 이번 부가세 X 룰로 합계가 원래 731,500 / 250,000 로 복귀 → 통장과 일치 자동 회복

**migrate.py 갱신**: 없음 (수식만 변경, formula_cols=[14, 15] 그대로 보호)

**docs/도메인_룰.md §1**: 매입 부가세 자동 분기 룰 추가
