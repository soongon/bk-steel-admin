## 권장 파일 (Claude.ai Project에서 가져와 여기 저장)

5월 시작 시점에 본인이 갖고 있는 컨텍스트 파일들을 이 폴더로 복사해둘 것:

- `최종_컨텍스트_요약.md` — 사업 전체 요약
- `친구_프로파일.md` — 친구 (SL철강 대표) 인적 정보, 강점·약점
- `자본구조_50대50.md` — 동업 자본 구조와 의사결정 룰
- `우리사업_시스템화_6개월_로드맵.md` — 5월~10월 단계별 계획
- `Claude_Project_시스템_인스트럭션.md` — Claude 프로젝트 운영 룰

## 사용 방법

이 폴더의 파일들은 시스템 코드와 무관. **Claude Code 사용 시에는 직접 참조 안 함**.

대신:
- Claude.ai Project Knowledge에 동기화 (수동 업로드 또는 sync 스크립트)
- 의사결정이 필요한 작업 (예: 가격 정책, 거래처 신용 평가, 분배금 계산) 시 Claude.ai에서 이 컨텍스트로 대화

## 갱신 주기

- 5월 매주 일요일: 그 주 동안의 의사결정/통찰을 `최종_컨텍스트_요약.md`에 누적
- 사업 구조 변경 시 (예: 직원 채용, 신규 거래처 대형 계약): 즉시 갱신
